## Installation
To get started with Form Api, follow these steps:

1. **Clone the repository**: 
    ```bash
    git clone https://github.com/sahil2k19/apiform.git
    ```


2. **Install dependencies**:
    ```bash
    npm install
    ```

3. **Set up environment variables**:
